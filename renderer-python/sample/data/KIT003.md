**ESP32 IoT Prototyping Kit** — rapid prototyping platform for connected devices.

Contents:
- 1x ESP32-WROOM-32 dev board with USB-C
- 1x DHT22 temperature/humidity sensor
- 1x BMP280 pressure sensor (I2C)
- 1x 0.96" OLED display (SSD1306, I2C)
- 1x WS2812B RGB LED strip (5 LEDs)
- Jumper wires, USB cable, quick-start guide

**Firmware examples** included for MQTT, HTTP REST, deep sleep and OTA update.
Fully compatible with ESP-IDF, Arduino core and MicroPython.
