**STM32 Development Starter Kit** — everything needed to get started with 32-bit ARM microcontrollers.

Contents:
- 1x STM32F103C8T6 "Blue Pill" board
- 1x ST-Link V2 USB programmer/debugger
- 1x 830-point breadboard + jumper wire set
- 1x USB-A to mini-B cable

**Getting started:** install STM32CubeIDE (free), connect ST-Link, flash the included blinky example.
Compatible with PlatformIO and Arduino core. Full documentation at parts.example.com/KIT001.
